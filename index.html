The Build

Shoulder Rules

  • No overhead pressing, pull-ups/chin-ups, pullovers, upright rows or dips.
  • Keep any raising movement at or below shoulder height.
  • If it pinches, cut the range or drop it.

Running Sessions

  • Easy / base: conversational pace (zone 2), can hold a chat.
  • Strides: easy run + 4–6 × ~20s relaxed fast bursts, full recovery between.
  • Intervals / VO2 / hills: the week's hard session — warm up well, run the efforts hard, jog recoveries.
  • Long run: easy pace, time on feet. Trail/undulating where you can for race specificity.

General fitness guidance, not medical advice — get the pressing checked by a physio given the bursitis, and ease off if anything flares.
